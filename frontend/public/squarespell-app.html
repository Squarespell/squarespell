Frequently asked questions

Can I cancel at any time?
Yes — cancel anytime from your account. No questions. Quizzes stay active until end of billing period.
How does the 7-day free trial work?
You get full access to your chosen plan for 7 days — no credit card required. After the trial, you can upgrade or stay on the free plan.
How does AI lead scoring work?
Our AI reads each lead's quiz answers and scores them Hot, Warm, or Cold based on urgency, budget, and engagement time. Hot leads are ready to buy now.
How is Cloudflare used in Squarespell?
We use Cloudflare Pages for hosting, Workers for our API layer, KV for fast data storage, R2 for file storage, and Access for secure authentication. Your data stays on edge servers closest to you.

Your embed code
<!-- Squarespell Quiz --> <div id="squarespell-quiz"></div> <script src="https://app.squarespell.com/embed/quiz-embed.js" data-quiz-id="YOUR_QUIZ_ID"> </script>

Quick install steps

1

Copy the code above

Click "Copy code" and replace YOUR_QUIZ_ID with your quiz ID from the Editor.

2

Add a Code Block in Squarespace

Edit your page → Add Block → Code. Make sure "Display Source" is off.

3

Paste, save, and publish

Paste the embed code into the Code Block, save your page, and you're live.

Tip: Your quiz will automatically match your Squarespace site's width. Place the Code Block wherever you want the quiz to appear.

Embed on Squarespace

Add your quiz to any Squarespace page in under a minute

Embed code
<!-- Squarespell Quiz Embed --> <div id="squarespell-quiz"></div> <script src="https://app.squarespell.com/embed/quiz-embed.js" data-quiz-id="YOUR_QUIZ_ID" data-accent="#D2FF1D"> </script>

How to add to Squarespace

1

Go to Squarespace Pages

Open your Squarespace dashboard and navigate to the page where you want the quiz.

2

Add a Code Block

Click Add Block → Code (or use the /code shortcut). Make sure "Display Source" is unchecked.

3

Paste the code

Paste the embed code above into the code block. Replace YOUR_QUIZ_ID with your actual quiz ID from the Editor tab.

4

Save and publish

Click Save, then publish your page. Your quiz is now live and collecting leads.

Tip: You can find your quiz ID on the Editor tab next to each quiz name. Each quiz has a unique ID you can embed independently.
